2. Concept d'ensemble

4 axes d'action :

  • 1 Rétrofit : équiper les engins existants de dispositifs de sécurité manquants
  • 2 Renouvellement accéléré : prioriser le remplacement des catégories critiques
  • 3 Spécifications achats : intégrer des critères de sécurité dans les cahiers des charges
  • 4 Signalétique renforcée : rendre les engins à risque plus visibles

5. Spécifications sécurité pour les achats

Créer un label "Sécurité GSE" porté par le CSCA :

  • Visibilité directe / caméras 360°
  • Détecteurs de proximité
  • ABS, freinage d'urgence
  • Éclairage LED
  • Limitation de vitesse paramétrable

Pondération proposée dans les appels d'offres : Sécurité 20%, Prix 40% (vs 50% actuellement).

6. Signalétique renforcée

Code couleur "Risque" :

Rouge critique : DÉGIVREUSE, PUSH>15 ans
Orange élevé : LOADER>15 ans, BUS>15 ans
Jaune moyen : GPU>15 ans
Vert faible : engins électriques récents

Coût signalétique renforcée : ~150 000€ (500 engins critiques).

8. Financement innovant
Mutuelle sécurité GSE
Fonds mutualisé handlers + ADP + DGAC pour financer 50-80% du rétrofit.
Leasing sécurité
Prestataire finance, handler paye un loyer mensuel.
Bonus/Malus à l'achat
Bonus pour engins équipés, taxe sur les non-équipés.

11. Conditions de réussite
  • Financement mutualisé - Le coût ne peut reposer sur les seuls handlers
  • Volontarisme initial - Un pilote réussi convaincra
  • Mesure de l'impact - Démontrer le bénéfice accidents évités
  • Standardisation - Privilégier équipements éprouvés
  • Maintenance - Assurer le suivi dans la durée
12. Avantages comparatifs
Pour le conducteur : visibilité améliorée, sécurité renforcée, conduite plus facile, fierté.
Pour l'entreprise : accidents évités, immobilisation réduite, image, attractivité.
Pour la plateforme : parc plus sûr, harmonisation des équipements.
